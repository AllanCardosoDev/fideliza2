# 🔧 SOLUÇÃO RÁPIDA: Bucket "documents"

## ⚠️ Situação

Você recebeu erro: **"Bucket not found"**

## ✅ Solução em 3 passos

### Passo 1️⃣ - Abra o Supabase

https://supabase.com/dashboard

### Passo 2️⃣ - Criar Bucket

1. **Storage** (menu esquerdo)
2. **Create new bucket**
3. **Nome:** `documents`
4. **Privacy:** `Private`
5. **Clique:** Create Bucket

### Passo 3️⃣ - Pronto!

1. **Volte para a aplicação**
2. **Recarregue:** F5
3. **Tente upload novamente** ✅

---

## ⏱️ Tempo Total: 2 minutos

Depois que criar o bucket, o upload funcionará normalmente!
